# RETIRED — legacy Vite frontend

This folder is the original `web/` React + Vite frontend that powered `basketch.vercel.app` before the v2.1 redesign.

**As of 2026-04-25 it is frozen.** All new frontend work happens in `web-next/` (Next.js 16, hosted at `basketch-redesign.vercel.app`).

## Do not modify

- No new features, no bug fixes, no dependency upgrades.
- CI no longer builds, tests, or type-checks this folder.
- Code is preserved here for git history / archaeology only.

## How to recover history

`git log --follow` still works on any file in this folder.
